# Online Auction System

A web-based auction platform developed using PHP and MySQL where sellers can list items and buyers can place bids on auction items.

## Features
- User Registration and Login
- Buyer and Seller Dashboards
- Sellers can add auction items
- Buyers can view items and place bids
- Highest bid wins the auction
- Secure session-based authentication

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- XAMPP

## Project Structure
- register.php – User registration
- login.php – Login authentication
- buyer_dashboard.php – Buyer dashboard
- seller_dashboard.php – Seller dashboard
- add_item.php – Seller adds auction items
- view_items.php – Buyers view auction items
- place_bid.php – Bid placement
- db.php – Database connection
